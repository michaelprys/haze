# Haze 🌄

> *Aesthetic Cloud Photo Gallery*

Live Demo: [hazeapp.netlify.app](https://hazeapp.netlify.app)

---

## Features

- Capture or upload photos, with optional location tagging.
- Update avatar, username, or bio instantly.
- Infinite scrolling feed with fast loading.

---

## Tech Stack

- Vue 3, TypeScript, Quasar, Pinia, SCSS  
- Supabase (PostgreSQL, Auth, Storage)

---
